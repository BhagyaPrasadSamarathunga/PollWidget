# Poll Widget

## Description

Implement a poll widget which can easily embedded into html pages.
Questions and answers are configurable.

<img src="https://github.com/BhagyaPrasadSamarathunga/Assets/blob/main/poll-widget.gif" width="900" />

## Setup

```
npm install
```

## Run App

```
npm start
```

## Run Tests

```
npm test
```
